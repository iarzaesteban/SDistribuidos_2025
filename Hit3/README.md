# Servidor TCP en Docker

## Descripción

Este proyecto implementa un servidor TCP en python usando docker y docker-compose.
Incluye pruebas automatizadas y un pipeline de CI/CD en GitHub Actions.

## Instalación y uso

### 1. Levantar el servidor

```sh
make up_server
```

### 2. Correr cliente

```sh
make run_client
```

### 2. Correr test

```sh
make test
```
